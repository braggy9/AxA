# AxA 🐉🎮

A top-down 2D adventure game built with Swift + SpriteKit.

**Designed by Zig (age 6). Built with Claude Code.**

## The Game
You play as an axolotl exploring worlds, solving puzzles, and battling evil axolotl knights and protectors. Unlock new playable characters with unique abilities. Obliterate the Dark Side.

## Worlds
1. **Salt World Lake** — Pink salt crystals, nono trees, and the giant mutant axolotl Monontoe
2. **Go-Go Forest** — Dense forest with secrets and vine-swinging (coming soon)
3. **The Underworld** — Volcanic stone, fire bushes, and Queen Owarla's temple (coming soon)

## Playable Characters
- **Wiz** — Wizard axolotl with a staff that becomes a grappling hook
- **Bob** — A chicken. Just a chicken. He's vibing.
- **Babeee** — Tiny baby axolotl who fits through small passages

## Tech
- Swift 6+ / SpriteKit / GameplayKit
- iOS 17+ (iPad primary, iPhone secondary)
- 16-bit pixel art
